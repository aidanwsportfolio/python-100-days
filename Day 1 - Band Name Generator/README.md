\# Day 01 - Band Name Generator



\## Project Instructions

Create a program that asks the user for the city they grew up in and the name of a pet, then combines the answers to generate a band name.



\## Concepts Practiced

\- Variables

\- User input

\- String concatenation

\- Printing



\## Potential Future Improvements
